# Responsive Web Layout - DecodeLabs Frontend Project 2

## What It Does
A fully responsive webpage using Mobile-First Strategy with:
- CSS Media Queries
- Flexbox for components
- CSS Grid for layout
- clamp() for fluid typography
- Popover API for mobile navigation

## Features
- Responsive navigation (hamburger menu on mobile)
- Grid layout that adapts: 1→2→4 columns
- Fluid typography with clamp()
- Clean, modern design

## How to Run
1. Open `index.html` in any browser
2. Resize the window to see responsiveness
