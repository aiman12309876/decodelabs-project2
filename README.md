# Backend API Development - DecodeLabs Full Stack Project 2

## What It Does
A RESTful API built with Flask that handles user data with full CRUD operations.

## Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | `/` | API welcome message |
| GET | `/users` | Get all users |
| GET | `/users/<id>` | Get user by ID |
| POST | `/users` | Add a new user |
| PUT | `/users/<id>` | Update a user |
| DELETE | `/users/<id>` | Delete a user |

## How to Run
1. Install: `pip install flask`
2. Run: `python app.py`
3. Test with Postman or browser
